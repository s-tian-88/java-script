WebPack

WebPack - это Module Bounder для JS-приложений, который позволяет обьеденить все ресурсы приложения в Bundle
По умолчанию брабатывает только JS-файлы и JSON
npm install --save-dev webpack - загрузка webpack
npm --save-dev install webpack-cli - работа с webpack через терминал

Загрузчики - позволяют Webpack обрабатывать другие типы файлов
npm <package-name>-loader - загрузчик

Точка входа - файл, с которого начинаются все действия webpack

Сборка - итоговый файл для испольъования в продакшене
